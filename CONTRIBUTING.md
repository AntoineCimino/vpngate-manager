## Contributing

Contributions are welcome.

You can help by:
- fixing bugs
- improving parsing reliability
- improving cross-distro compatibility
- enhancing documentation
- suggesting safer defaults
- adding tests

## Development notes

- Please keep changes small and focused.
- Do not commit generated artifacts (`*.ovpn`, `vpn.log`, `vpn.pid`, `.vpngate_cache.csv`).
- Run `shellcheck` on `vpngate-manager.sh` when possible.
